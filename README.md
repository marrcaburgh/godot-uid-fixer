# godot-uid-fixer
Fixes the UID error in godot by randomizing the UIDs in a file.

## Prerequisites
CMake

C++ Compiler

## Build Instructions
`cd` to root directory of repo.

Run `cmake .`

Run `make`.

## Installation instructions
Arch Linux: download and install [godot-uid-fixer-git](https://aur.archlinux.org/packages/godot-uid-fixer-git)
