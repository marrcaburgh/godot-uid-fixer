# godot-uid-fixer
Fixes the UID error in godot by randomizing the UIDs in a file.

## Build instructions
### Prerequisites
CMake

C++ Compiler

### Building
`cd` to root directory of repo.</br>
Run `cmake .`</br>
Run `make`.</br>

## Installation instructions
### Arch Linux
download and install [godot-uid-fixer-git](https://aur.archlinux.org/packages/godot-uid-fixer-git)
